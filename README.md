# Twitter Sentiment Analysis using RNN

## Overview

This project builds a Recurrent Neural Network (RNN) to classify tweets into:

* Positive
* Neutral
* Negative

## Features

* Text preprocessing
* Tokenization
* Padding sequences
* RNN model using SimpleRNN

## Technologies Used

* Python
* TensorFlow / Keras
* Pandas
* NumPy

## Dataset

Twitter dataset (~20,000 tweets)

## Model Architecture

* Embedding Layer
* SimpleRNN
* Dense Layer
* Softmax Output

## Results

Achieved strong validation accuracy.
